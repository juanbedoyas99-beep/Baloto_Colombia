# 🎰 Baloto Colombia Data Analysis (2017–Present)


## 📌 Project Overview

This repository contains an exploratory and statistical analysis of the **Baloto Colombia lottery results** from **2017 to the present**.

The dataset was obtained from Kaggle:
👉 https://www.kaggle.com/datasets/jaforero/baloto-colombia

The objective of this project is to analyze numerical patterns, frequency distributions, positional behavior (C1–C5, Superbalota), and statistical characteristics of the draws over time.

This repository focuses on structured data exploration and visualization to better understand the probabilistic behavior of lottery outcomes.

---

## 🧠 Methodology

1. **Data Collection**  
   The dataset was downloaded from Kaggle and includes historical draw results starting in 2017.

2. **Data Cleaning & Structuring**  
   - Validation of draw records  
   - Standardization of numerical formats  
   - Separation of positions (C1–C5, SB)  
   - Creation of frequency tables  

3. **Exploratory Data Analysis (EDA)**  
   - Frequency analysis per position  
   - Global frequency distribution  
   - Positional comparison (C1 vs C2 vs C3…)  
   - Temporal evolution of numbers  
   - Detection of clustering or distribution bias  

4. **Visualization**  
   Multiple graphical representations were generated, including:
   - Bar charts  
   - Scatter plots  
   - Heat maps  
   - Box plots  
   These visualizations help identify patterns in number occurrence and positional behavior.

---

## 📈 Key Objectives

- Analyze number frequency since 2017  
- Compare positional distributions (C1–C5 & Superbalota)  
- Evaluate dispersion and central tendency  
- Explore statistical randomness behavior  
- Provide structured datasets for further probabilistic modeling
